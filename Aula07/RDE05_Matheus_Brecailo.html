Hiperlinks

Hiperlinks conectam informações de um documento de hipertexto.

Estas referências podem ser :

  • Interna
  • Local
  • Arquivo
  • Remoto
  • Endereço de Correio

Sintaxe Geral

A marcação de links acontece utilizando a TAG e href que faz referência ao conteúdo desejado

Hiperlinks Externos ou Remotos

São links que direcionam o usuário a outra páginas web. Utiliza-se URL (Uniform Resource Locator) para a referência que desejamos.

Hiperlinks Internos

São links que direcionam o usuário a um determinado local da mesma página web

Hiperlinks para Endereço Eletrônico

São links que direcionam para um endereço eletrônico
