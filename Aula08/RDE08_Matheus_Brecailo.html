Algumas tags uteis

< table >

A tag < table > define uma tabela HTML.

Uma tabela HTML consiste em um elemento<table> e um ou mais elementos < tr > , < th > e < td > .

  • < tr > : define uma linha da tabela.
  • < th > : define o cabeçalho da tabela.
  • < td > : define uma parte da tabela.

< del >

A tag < del >define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.

< ins >

A tag < ins > define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Exemplo:

Helo WOrd
Hello Word

< sub >

A tag< sub > define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor.

Exemplo:
H2O

< sup >

A tag < sup > define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor.

Exemplo:
24 = 16

< blockquote >

A tag < blockquote > especifica uma seção que é citada de outra fonte.

Exemplo:

Como dizia Thomas Jefferson Quando o mundo estiver unido na busca do conhecimento, e não mais lutando por dinheiro e poder, então a nossa sociedade poderá enfim evoluir a um novo nível.

< q >

A tag < q > define uma cotação curta.

Exemplo:

Como dizia René Descartes Penso logo existo.

< abbr >

A tag < abbr > define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Exemplo:

O atual campeão da UEFA é o Real Madrid da Espanha.

< address >

A tag < address > define as informações de contato do autor/proprietário de um documento ou artigo.As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, identificador de mídia social, etc.

< cite >

A tag < cite > define o título de um trabalho criativo (por exemplo : um livro).

< bdo >

A tag < bdo > é usada para substituir a direção do texto atual(ao contrário).

Exemplo:

Subi no ônibus

< code >

A tag < code > é usada para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

< pre >

A tag < pre > define o texto pré-formatado.

< kbd >

A tag < kbd > é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Exemplo :

Press Ctrl + C to copy text (Windows).

Press Cmd + C to copy text (Mac OS).
